
# Chronic Kidney Disease Dialysis Prediction

![Screenshot](./screenshot_ckd.png)

Libraries:  
  
* streamlit~=1.10.0
* pandas~=1.2.5
* numpy~=1.19.5
* matplotlib~=3.4.2
* shap~=0.40.0
* scikit-survival~=0.17.2
 
Run:

 2. All models will be saved to pickle files
